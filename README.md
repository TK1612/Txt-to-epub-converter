# 📚 Novel to EPUB Converter

![649178036_950205537550061_276526426078080711_n](https://github.com/user-attachments/assets/6f7abce4-5bd4-434f-8828-5a20f86740b7)

A powerful, user-friendly Streamlit web application designed to convert raw novel text files (like raw KR text dumps) into beautifully formatted EPUBs and HTML chapters. 

Whether you're processing web novel translations, cleaning up raw text dumps, or just organizing a massive text file into a readable format, this tool automates the heavy lifting. It features regex-based chapter extraction, real-time table of contents editing, and visual HTML previewing.

## ✨ Features

* **Intelligent Chapter Extraction:** Upload a `.txt` file and let the app scan and parse chapters automatically.
* **Sequence Scanning:** Automatically detects and alerts you of any missing chapters in the numeric sequence so you can catch formatting errors early.
* **Editable Table of Contents:** Review and rename extracted chapter titles directly in the browser before building your files.
* **Advanced Regex Formatting:** Apply built-in or custom Regular Expressions to clean up HTML tags, handle pesky blank `<p>` tags, and perfectly format your headers.
* **Live Rendered Previews:** Compare raw HTML code with a visual rendered preview of your chapters to see exactly how your regex rules affect the final output.
* **Custom Metadata & Covers:** Add custom book titles, authors, language codes (defaults to `ko`), and upload your own cover images.
* **Dual Output Formats:** Download the final product as a polished `.epub` file or a `.zip` archive containing individual HTML chapters.
